<b>
Overview</b>
</div>
<div>
<div>The Principle Network Engineer will be responsible for designing, ordering, configuring, and implementing data telecommunications networks.</div>
<ul>
<li>Has responsibility for design, planning plus delivery Local Area Network (LAN), Wide Area Network (WAN) product solutions using broadband, wireless (4G-3G-2G, Wi-Fi, Zigbee - IOT) and VSAT.
</li>
<li>Work with network architect to direct network engineering within the US/ Canada region.
</li>
<li>Provides senior-level expertise in the design, integration, and management of data networks.</li>
<li>Integrates project objectives with overall business strategies.</li>
<li>Implements network standards to support the global communications networks.</li>
<li>Team member for delivery of LAN product releases to technology group.</li>
<li>Oversees testing and evaluation of network communications devices both physical and virtual for product release. This includes routers, application delivery controllers, switches, firewalls and addition security such as intrusion prevention systems.</li>
<li>Hands on experience using command line and GUI interaction with Fortinet, Cisco, Juniper and A10 hardware/ software.
</li>
<li>Implements Network Management tools to monitor/evaluate network performance with knowledge of SNMP using Zabbix and Solarwinds.</li>
<li>Assists in providing solutions to reduce costs continuning to ensure optimal network performance.</li>
<li>Supports delivery when requested to fine tune production networks as well as level 3 escalation.</li>
<li>Provide subject matter expert support for operational incidents when requested</li>
<li>Provides training for network delivery and operations to assist them in the performance of their assigned duties operating gaming networks.</li>
<li>Provides technical expertise to both internal and external customers for pre-sales and some contract negotiation.</li>
<li>Ideally experience using SDWAN, AWS networking and SDN to support latest initiatives.</li>
<li>Network automation for product releases to delivery using Ansible.</li>
<li>Network configuration control using GIT.</li>
</ul>
