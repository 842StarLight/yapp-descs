<div><b>JOB SUMMARY</b></div>
<br>
<div></div>
<div>The Senior Software Development Engineer (SDE) is a technical contributor role responsible for full stack application development within the Infrastructure Domain at Marriott International. SDEs within the Infrastructure organization are responsible for the design, development, delivery, deployment, and maintenance of applications within the Marriott International Cloud Platform (MICP). Candidates for Infrastructure SDEs should be experienced with full stack application development which includes (and is not limited to), application and database design/architecture, code development, artifact compilation, pipeline development and management, code repository management, Kubernetes, containerized applications, and systems operations.</div>
