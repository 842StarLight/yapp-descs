<p>
Summary of Role<br>
The Security Analyst III is responsible for creating, updating, maintaining, and documenting, security configuration and profiles in several major information systems used throughout the business. This role will make recommendations to the IR Security Manager for security policies, standards, guidelines, and procedures along with identifying and mitigating possible vulnerabilities throughout CROPP’s systems. The Security Analyst III will also be responsible for monitoring and reporting security exceptions.</p>
<p>
Essential Duties and Responsibilities</p>
<ul>
<li>
Investigate and respond to security incidents.</li>
<li>
Work collaboratively in the Security Operations Center.</li>
<li>
Identify root cause of security incidents and provide mitigation.</li>
<li>
Develop and maintain user security profiles for CROPP’s major business systems.</li>
<li>
Responsible for all client and application security configuration.</li>
<li>
Lead the effort along with IT and Business System owners to establish security roles.</li>
<li>
Provide continuous monitoring of CROPP’s security controls.</li>
<li>
Organize and facilitate interdepartmental process meetings to clarify security issues.</li>
<li>
Create and maintain documentation for security change control.</li>
<li>
Document and explain deviations from standard security profiles.</li>
<li>
Make recommendations regarding security requests.
</li>
<li>Field, investigate and respond to requests and questions from all areas of the business related to security.</li>
<li>
Implement new security systems and changes to existing systems.</li>
</ul>
<p>
Additional Duties and Responsibilities</p>
<ul>
<li>
Participate in special projects throughout the business as assigned.</li>
<li>
Present security implementations and changes to CIO and IR Department Leadership.</li>
<li>
Other duties as assigned by manager.</li>
<li>
Handle confidential information with discretion.</li>
</ul>
