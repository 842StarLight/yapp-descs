<div>Job Summary: Serves as the application administrator for a variety of Store System (SS) software applications running on a variety of hardware environments.<br>
</div>
<div></div>
<div>Incumbents of this position must be U.S. Citizens.<br>
</div>
<div></div>
<div>Duties and Responsibilities
</div>
<ul><li>Serves as liaison between the helpdesk, systems administrators, database administrators and functional (Store) users.
</li></ul>
<ul><li>Installs and configures STORE SYSTEMS application software.
</li></ul>
<ul><li>Sets/manages application parameters for all STORE SYSTEMS.
</li></ul>
<ul><li>Point of contact between application vendors, database administrators and systems administrators for application functionality and configuration.
</li></ul>
<ul><li>Troubleshoots technical STORE SYSTEMS applications problems and coordinates resolution of problems with other support groups.
</li></ul>
<ul><li>Assists in implementation of security strategy for the application based on requirements.</li></ul>
<ul><li>Monitors STORE SYSTEMS applications for performance problems and coordinate resolutions with other support groups.
</li></ul>
<ul><li>Obtains regular processing reports to check for errors, assess/analyze errors and distinguishes between databases, application and customized application problems.
</li></ul>
<ul><li>Monitor, coordinate and install vendor provided Store Systems applications software patches, cyclical releases and major upgrades.
</li></ul>
<ul><li>Assist with developing stategy for the installation of major releases.
</li></ul>
<ul><li>Assist in providing supporting documentation for Store Systems.
</li></ul>
<ul><li>Perform impact analysis on all patches prior to installation.
</li></ul>
<ul><li>Provides technical support for interfaces.
</li></ul>
<ul><li>Respond to Service Desk Requests within the parameters set forth by NEXCOM SLA's.
</li></ul>
<ul><li>Ensure/implement the rigorous application of Information Security/Information Assurance policies, principles, and practices in the delivery of Systems, Applications and/or Services (Hardware &amp; Software).
</li></ul>
<ul><li>Performs other related duties as assigned.<br>
</li></ul>
