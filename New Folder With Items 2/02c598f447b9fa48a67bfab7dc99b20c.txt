<div>Network, Desktop, and Application Support Administrator, SQL Server Experience Highly Desired<br>
<br>
We're looking for an experienced network, desktop, and application administrator to support both our onsite and work-from-home virtual sales force. The ideal person is ready to train and develop a hungry Junior IT Administrative Support staff and delegate most of the routine responsibilities outlined below so s/he can grow into a new role expanding his/her SQL administration knowledge, skills and abilities towards a SQL Server DBA.<br>
<br>
<b>Our network support</b> environment includes 11 windows (vm) servers, 4 Vmware hosts, 5 Linux hosts (web, mongo, storage, google sync), 2 High-availability firewalls (DC and office), 2 Mikrotik routers, 1 Storage(backup) host, 1 core managed Cisco switch, 3 Netgear Smart switches (office), 2 wireless access points (office). If you have comparable knowledge with competing systems we are open to transitioning from any of the above to an alternative.<br>
<br>
<b>Our desktop support </b>environment includes 18 office desktops, 14 remote worker desktops, and associated routers / printers.<br>
<br>
<b>Our application support</b> includes SQL Server, Office365</div>
