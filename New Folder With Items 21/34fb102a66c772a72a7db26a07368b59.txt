<p><b>REQUIREMENTS</b></p>
<p>Here’s Some Of The Skills We’re Looking For:</p>
<ul>
<li>Familiarity with design patterns (GOF), Android Architecture Components, Clean Architecture principles, and RxJava.</li>
<li>Kotlin as well as Java.</li>
<li>Developing complex applications that integrate with back end services.</li>
<li>Developing for a large install base (we have 10 million+ installs).</li>
<br>
</ul>
<p></p>
<p>What this job entails</p>
<ul>
<li>Writing maintainable, testable code and unit tests.</li>
<li>Working with new and legacy code.</li>
<li>Implementing new features and improvements as well as fixing bugs and improving performance.</li>
<li>Working with cryptography libraries.</li>
<li>Giving and accepting feedback in code reviews.</li>
<li>Making use of continuous integration to automate internal builds, unit tests, release builds, etc.</li>
<li>The opportunity to attend Google I/O.</li>
<br>
</ul>
