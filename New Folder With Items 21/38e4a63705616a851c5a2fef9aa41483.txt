<b>Introduction:</b><br>
<br>
Toshiba Global Commerce Solutions is seeking a Software Engineer IV to work as a key member of our Next Generation Platform Transaction Engine development team. This indvidual will write Java and Javascript code, design and implement RESTful microservices and platform infrastructure, develop javascript administration uis for services, integrate services to frontend javascipt sales uis, assist junior developers with code implementation and design, and (on a rotating basis with other senior developers on the team) act as scrum master on daily scrum and sprint closeout meetings.<br>
<br>
<b>Responsibilities:</b><br>
<ul>
<li>Design and Implement Java8 Springboot RESTful microservices</li>
<li>
Design and Implement VueJS Administration UIs for RESTful services</li>
<li>
Design and Implement Javascript WebPOS UI integrations to RESTful services</li>
<li>
Design NoSQL database collections, indexes, and data models</li>
<li>
Design and Implement RESTful endpoints using OpenAPI3/Swagger</li>
<li>
Implement JUnit regression tests</li>
<li>
Implement Selenium BDD tests for services and Javascript UIs</li>
<li>
Lead daily scrum and sprint closeout meetings</li>
<li>
Contribute to backlog grooming and design</li>
<li>
Assist junior team members on a daily basis with design and implementation</li>
<li>
Updating and managing VersionOne backlog stories/tasks/defects/etc</li>
<li>
Maintain quality and ensure responsiveness of the system</li>
</ul>
<br>
<b>
 </b><b>Required Skills:</b><br>
<ul>
<li>Degree in Computer Science or related field or equivalent experience</li>
<li>
At least 10 years' experience as a Full Stack developer using Java and Javascript</li>
<li>
Experience as a team lead, senior developer, or other senior/leadership role on a professional software development project</li>
<li>
Able to read, write and speak English proficiently and be an effective verbal and written communicator.</li>
<li>
Self-sufficient and maintaining development environment and process to maintain project code integrity and organization</li>
