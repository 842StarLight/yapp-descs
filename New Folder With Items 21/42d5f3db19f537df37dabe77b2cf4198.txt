<h2>Web Developer Co-Op Spring 2020</h2>

<b>POSITION SUMMARY:</b><br>
The Web Developer Intern is responsible for the development, integration, and implementation of web-based solutions that support our overall company business strategies and objectives. As part of this internship, you’ll learn how to develop and test code efficiently while gaining real world experience working on web and mobile projects that support corporate marketing initiatives. With mentoring from the Senior Java Web Developers, the intern will learn multiple systems and be able to use skills in multiple projects and maintenance.
<br><br>
<b>KEY RESPONSIBILITIES:</b><br>
Developing and testing code for web and mobile applications using modern methods of development across the full stack (frameworks, web services, etc.)
<br>Scope basic project requirements
<br>Troubleshoot and resolve application issues
<br>Interact with team members (end-users, testers, distributors, analysts and project/release managers) in a team-oriented fashion
<br>Support projects through entire Software Development Lifecycle (SDLC)
<br>Follow software development methodologies and best practices
<br>Provide end-user application support
<br>Adhere to web security best practices
<br>Ability to adapt technologies to meet the needs of Matco Tools in a manner that can be easily utilized across the enterprise
<br>Work on additional projects as needed in the Marketing Department
<br>Present to Executives and Key Managers at the end of co-op on projects, learnings, and overall experience
<br><br>
Attributes/Skills/Experience
<br><br>
Must be a current Junior or Senior pursuing a Bachelor’s degree in Computer Science, Information Systems, or a related field
<br>1+ year prior internship or classroom experience with Java programming, various web technologies such as HTML, JavaScript, CSS, Web Service APIs, Design Frameworks and SQL or RDBMS design
<br>Desire to learn and adapt to rapidly changing technologies
<br>Excellent communication skills, written and oral
<br>Ability to work collaboratively and effectively with other IT teams
<br>Knowledge of Java application development
