<h2>Cloud Architect</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText">
<div>
<p>Nutanix is a cloud computing software company that sells hyper-converged infrastructure appliances and software-defined storage. Founded in 2009, we have been a publicly held company since 2016. Named 2018 Gartner Magic Quadrant for Hyper-Converged Systems.<br>
<br>
Nutanix is the right fit for you if you are passionate about technology, creative, and accomplished in your growing career. We pride ourselves on our encouraging culture where you learn from failures, pick yourself back up and become the best version of yourself. Nutanix offers several opportunities to participate in professional development and learning, fostering an inspiring environment. We believe in not only supporting you, but also your families through fully-paid healthcare plans, equity, and other financial tools and wellness programs.</p>
<div></div>
<br>
<div>
Nutanix Services and Customer Solutions (S&amp;CS) primary goal is to ensure the success of our customers. The Services function engages customers in the pre-sales and post-sales stages to deliver high-valued outcomes in their software-defined data center (SDDC) strategy and architecture. Besides helping customers implement Nutanix products successfully, the Services team, working with our partners, delivers migrations, hybrid cloud, VDI, and Tier 1 Application re-platforming solutions to enable truly software-defined data centers. The Nutanix Services function includes consulting, planning, design, implementation, integration and automation services as well as project management.</div>
<div></div>
<br>
<div>
The Cloud Consulting Architect will work under the Services team to create service offerings to enable our customers to deploy automation and orchestration solutions provision to multiple cloud end points (e.g. Nutanix, AWS, Azure, GCP, etc). He or she will work with software developers, consultants, and engineers to create scalable and reliable solutions for clients and partners.<br>
</div>
<div></div>
<div>
<b>Responsibilities:</b><br>
</div>
<ul>
<li> Billable Project work 70% of time</li>
<li> Work with Nutanix customers to understand business goals and translate them into cloud automation and orchestration solutions
</li>
<li> Evangelize Nutanix’s vision for a multi-cloud world and offer hands on experience delivering these outcomes for our customers</li>
<li> Run customer facing workshops to develop the Cloud requirements and design and document these for customer review and approval
</li>
</ul>
<div>Design service catalogs for Self-Service Provisioning (SSP) leveraging the Nutanix Calm Automation and Orchestration platform to provisioning and manage mode 2 applications delivered to both private cloud and public cloud end-points<br>
(note: Nutanix Calm experience not required and can be learned on the job)</div>
<ul>
<li> Design services that leverage cloud native architectures, microservices like Docker and Kubernetes, as well as IaaS and PaaS architectures
</li>
<li> Develop detailed blueprint specifications that developers will use to write code and develop the automation components. Integration with 3rd party software products like (Infoblox, Active Directory, Service Now, etc) via RESTAPI is integral to this work.</li>
<li> Work collaboratively in a scrum agile team leveraging a variety of DevOps tools (e.g. Jira, GitHub, Jenkins, etc) as part of the CD/CI pipeline</li>
<li> Work with software development teams at a hands-on level to design and write code</li>
<li> Develop and test end-to-end blueprint incorporating all developer code to deliver the desired outcome
</li>
<li> Participate in a limited pre-sales involvement to provide services overlay to the existing sales motion primarily as a SME</li>
<li> Create toolkits and best practice documents and whitepapers<br>
</li>
</ul>
<div></div>
<div><b>Required Qualifications:</b></div>
<ul>
<li> Up to 70% of travel
</li>
<li> A minimum of 5 years of industry experience</li>
<li> Excellent architecture expertise with in-depth understanding of cloud architecture patterns</li>
<li> Ability to whiteboard solutions interactively with customers and develop robust architectural documentation</li>
<li> High comfort level using one or more automation tools or orchestration application stacks and frameworks (e.g. Terraforms, Lambda, Azure Pack, Azure Stack, OpenStack, Cloud Stack, VMware vCO, or Microsoft Systems Center Orchestrator)</li>
<li> Experience with developing and managing cloud-computing environments like Amazon EC2, Azure and/or GCP</li>
<li> Experience developing scalable cloud microservices using container technologies like Docker, Kubernetes, and Mesosphere
</li>
<li> Ability to develop and debug software scripting and code in at least one of the following languages (Powershell, Bash, Python, Perl, Ruby, Java, Javascript, etc.)
</li>
<li> Understanding of architecture and use cases for configuration management tools like Puppet, Chef, Ansible, and SALT Stack</li>
<li> Leverage DevOps tools like GitHub, Jira, Jenkins, Artifactory, Zephyr, etc. to deliver automation and promote a DevOps culture</li>
<li> Experience with building and running large scale, distributed, highly-available, fault-tolerant, multi-tier applications running on both Windows and Linux</li>
<li> Ability to manage multiple projects while maintaining a high level of detail</li>
</ul>
