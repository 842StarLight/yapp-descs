<div><b>
Description</b></div>
<div>As an Application Analyst/Developer, this individual will code software applications for pipeline operations and conduct planning that provides critical operations support for client companies.<br>
</div>
<div></div>
<div><b>Responsibilities</b></div>
<div>
</div>
<ul>
<li><p>Delve into existing code as well as write new applications.
</p></li>
<li><p>Fundamental product analysis and design, source code development, and project driven development.</p></li>
<li><p>
Manage a total software solution for pipeline scheduling.
</p></li>
<li><p>Organize and consolidate information required to generate short- and long-term schedules for liquid pipeline movements.
</p></li>
<li><p>Help pipeline companies schedule their pipelines efficiently and rapidly within a single application.
</p></li>
<li><p>Allows the user to rapidly collect shipping requests and create a working schedule and manage that schedule through its active period. Schedules can be created based on nominated volumes or management of terminal inventories, or any combination of these.</p></li>
<li><p>
Determine coding design requirements from function and detailed specification.</p></li>
<li><p>
Analyze software bugs and affect code repairs.</p></li>
<li><p>
Design, develop, and deliver specified software features.</p></li>
<li><p>
Produce usable documentation and test procedures.</p></li>
<li><p>
Deal directly with the end clients to assist in software validation and deployment.</p></li>
</ul>
