<h2>Production Engineer</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText"><div>
<p><b>Position Title</b>: Production Engineer</p>
<br>
<p></p>
<p><b>
Reports to</b>: Production Manager</p>
<br>
<p></p>
<p><b>
Primary Function</b>:<br>
</p>
<p></p>
<p>The primary function of Production Engineer is provides an in-depth hardware / software technical and technological expertise in their particular areas of focus, develop / provide a genuinely customized solution for Production.
</p>
<p></p>
<p><b>Duties and Responsibilities:</b></p>
<br>
<p></p>
<p>
· The applicant will be responsible for new product evaluation; customer project configuration file (CTO-Configure-to-Order file) review and update, system integration and functional test for all PC related products. This requires a strong knowledge of hardware and software, as well as the technical skill in various MS operating systems and network connectivity for all computer station and server systems. Knowledge in Linux OS is also preferred. Knowledge in UNIX is plus.
</p>
<p>· Strong capability in CTO (Configure to Order file) and SOP (Stand Operation Procedure) setup and modification is required.</p>
<p>
· New project (hardware / software) evaluation, implementation and training to teams in a timely manner before mass production.</p>
<p>
· Responsible in new system integration for all computer units (Advantech system / CPU board), including hardware inspection, OS installation, power on test, functional test and trouble-shoot in the system function level when needed.</p>
<p>
· Perform a hardware and software level instructor in Production. PE provides the system rework, upgrade, modify, test training to all production staffs.</p>
<p>
· Work and interface with AE groups and Sales teams, initiate, create CTO file and provide the right response / solution positively and professionally to solve the system functional issues and configuration problem.</p>
<p>
· Good communication and writing skill is also required</p>
<p>
· Accomplishing all tasks assigned to you</p>
<br>
<p></p>
<h4 class="jobSectionHeader"><b>
Skills/Education requirements:</b></h4>
<br>
<p></p>
<p>
Ø 2 yrs degree, MCSE or above in a technical/scientific field or equivalent experience.
</p>
<p>Ø Completion of formal technical training (3+ years)</p>
<p>
Ø Product-oriented certificates
</p>
<p>Ø Minimum 2 years hand-on / professional experience in PC hardware integration areas.
</p>
<p>Ø Ability to handle multiple tasks, and a time management skills a must.</p>
<p>
Ø Ability to manage a changing workload / priorities, meet order schedule with quality results.</p>
<p>
Ø Ability to work independently, analytical problem-solving skills.<br>
Ability and willingness to motivate and support others and team player</p>
<p>
Ø Clear verbal and written communication skills</p>
