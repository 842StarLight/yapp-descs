About Authentic8
<br>Founded in 2010 by principals from Postini (acquired by Google in 2007), Authentic8 is redefining how the browser is used to access sensitive web data. Silo, our flagship product, is a cloud-based remote browser. Silo secures access to sensitive sites, creating an insulation layer between the computer and the web service. Silo helps manage login credentials, access controls and data use policies. Silo is built fresh at session start, and destroyed at session end, ensuring that users remain secure, compliant, and anonymous online. Learn more at www.authentic8.com.
