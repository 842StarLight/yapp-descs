<h2 class="jobSectionHeader"><b>Responsibilities &amp; Qualifications
</b></h2>
<p></p>
<p>The Java Enterprise Edition Developer is responsible for designing, modifying and developing systems using advanced techniques including the ability to analyze current systems to resolve new and/or existing problems. The developer evaluates technical feasibility of proposed system solution, prepares project proposals and develops/modifies systems to meet the business need. The developer should have experience delivering scalable, secure, and robust web applications. They will act as resource for colleagues with less experience; may lead or direct the work of other developers/projects. The developer further cultivates analysis and development skills by learning new and emerging technologies.
</p>
<p></p>
<p><b>ESSENTIAL SKILLS
</b></p>
<ul>
<li>Ability to design and implement scalable, secure, highly available applications
</li>
<li>Excellent verbal and communication skills
</li>
<li>Excels in ability to multi-task and stay organized in a dynamic work environment.
</li>
<li>Excellent interpersonal skills with the ability to work well with subordinates, peers and management.
</li>
<li>Designing, modeling and documenting program/code for an integrated software product or system that supports re-use, change and integration of various components
</li>
<li>Constructing software; that meets requirements, promotes re-use, is testable and facilitates ease of support
</li>
<li>Conducting unit tests, tracking problems, and implementing changes to ensure adherence to test plans and functional requirements
</li>
<li>Diagnosing, isolating, and implementing remedies for system failures caused by software code, using knowledge of the system characteristics and utilizing system maintenance and tracking tools
</li>
<li>Willingness to learn new technologies, such as VueJS, Node, etc<br>
</li>
</ul>
<p></p>
<p><b>EXPERIENCE
</b></p>
<ul>
<li>College degree in Information Technology (preferred)
</li>
<li>Minimum of 5 years work experience in Information Technology
</li>
<li>Minimum of 4 years Java/ J2EE (including Servlets/JSPs), JDBC, HTML, JavaScript, CSS, SQL and XML development experience, including Java IDE’s such as Eclipse or RAD.
</li>
<li>Minimum 2 years frame work experience in STRUTS, SPRING, or similar JavaScript frame works.
</li>
<li>Minimum 2 years’ experience developing REST Services using Spring MVC / JAX-RS and/or experience building and consuming JAX-WS / JAX-RPC Web Services
</li>
</ul>
