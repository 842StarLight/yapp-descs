<b>Job Summary:</b>
</div>
<div><div>Supermicro is looking for talented and passionate Application Engineer who has strong software programming skills to support onsite manufacturing and customer technical support for networking, switch or server products. You will be responsible in providing technical supports to external and internal customers. The ideal candidate must have strong technical background in software engineering, hardware architecture, networking as well as exceptional data analysis skillset for software automation.

<ul><li>Responsible for post sales on-site technical support, provide optimized solutions for customers using company's Server, Networking or Switch products</li></ul>

<ul><li>Handle day-to-day customer technical support of networking related issues including switches, add-on-cards and Server Products</li></ul>
