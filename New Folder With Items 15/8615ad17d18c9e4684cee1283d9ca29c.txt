<h2>Solution Manager, Core Business Operations, Systems Engineering</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText"><div>
<div>Deloitte Consulting LLP seeks a Solution Manager, Core Business Operations, Systems Engineering in Gilbert, AZ and various unanticipated Deloitte office locations and client sites nationally.</div>
<div>
<br>
</div>
<div><b>Work You’ll Do</b></div>
<div>
<br>
</div>
<div>Managing day-to-day interactions with executive clients and sponsors. Managing and delivering components of client engagements that identify, design, and implement technology and creative business solutions for large companies. Responsibilities include, managing teams in the identification of business requirements, functional design, process design (including scenario design, flow mapping), prototyping, testing, training, defining support procedures. Developing and maintaining contact with top decision makers at key clients. Participating and leading aspects of the proposal development process. Contributing to the development of proposal pricing strategies. Performing role of counselor and coach; actively participating in staff recruitment and retention activities. Performing assessments of existing information systems, and identifying gaps between business requirements and current system capabilities. Estimating time and resources needed for the development work. Creating project charters, schedules, and plans, defining project tasks, allocating resources and monitor project progress. Performing conceptual and technical designs, including the use of object-oriented analysis and design techniques. Working on E2E (Exchange to Exchange) from requirement gathering, project sizing, team onboarding, defining to code review and project management. Overseeing project management using Mainframe, MS.NET (Microsoft.net), JAVA and BIG DATA technologies. Working on the complete AGILE (Rally and JIRA), Kanban and Waterfall SDLC (System Development Life Cycle) processes define, analysis, design, development, testing and production support of applications. Using Mainframe technologies JCL (Job Control Language), COBOL (common business-oriented language), VSAM (Virtual Storage Access Method), DB2 (Database 2), IMS (Inventory Management System), QMF (Query Management Facility), SPUFI (Structured Query Language Processing Using File Input), File Aids, FM (File Manager), MQC (Mercury Quality Centre), EZYTRIVE, Job-Track, CONTROL-M, Sort-Cards PL-SQL, IBM OPTIM, and Native Store Proc. Working on Service now and Manage now, using big data technologies HDFS (Hadoop Distributed File System), HIVE, PIG, HBASE (Hadoop Database), SQOOP (SQL-to-Hadoop). Utilizing Java technologies Advance JAVA, Web Services, Framework SPRING, Tool JDK (Java SE Development Kit), Maven, Eclipse, Application Server JBOSS (JavaBeans Open Source Software), Tomcat, and Platform PaaS (Platform as a Service).</div>
