<h2>IT Application Support Specialist</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText"><div>
<p><b>Objective:</b> Answer queries and resolve issues with software applications. Train users on specific applications. Answer questions and resolve computer problems for internal users. Duties include providing assistance concerning the use of computer hardware and software, including printing, installation, electronic mail, and operating systems for all Sheehan Family Companies. Support includes off hours availability and travel to remote sites (20%).</p>
<p></p>
<br>
<p><b>Regular Working Hours: Monday- Thursday 11am-8pm, Friday 9am-6pm</b></p>
<p></p>
<p><b>Responsibilities:</b></p>
<ul>
<li>Provide support for Company's Route Accounting System (VIP Beverage).</li>
<li>Provide support for warehouse management systems and operations technologies including but not limited to iPhones, RF Units, Voice Pick equipment.
</li>
<li>Provide support for Mobile Applications, POS applications, Sales Reporting, Retailer Portal and CRM System.</li>
<li>Support additional company used applications including Roadnet/Mobilecast, Nexus, PrismHR, and other regularly used software applications.</li>
<li>Set up equipment for employee use, performing or ensuring proper installation of cables, operating systems, user set up and appropriate software and hardware. This includes but is not limited to: personal computers, printers, and workstation. Have an understanding of end user job function to ensure all appropriate end user needs are installed and readily available.</li>
<li>Refer major hardware or software problems or defective products to vendors or technicians for service. Install and perform minor repairs to hardware, software, or peripheral equipment, following design or installation specifications.</li>
<li>Have an understanding of the servers and the components that make up the network. Maintain basic troubleshooting skills for server and network issues. Know when to contact network vendors or internal server and network support staff</li>
<li>Travel to remote sites to offer support, training and project leadership or assistance.</li>
<li>Provide back up support for system access, email access and user security Maintain a professional and support driven disposition with end users</li>
</ul>
