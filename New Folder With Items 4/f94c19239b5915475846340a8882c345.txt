<h2>IT Security Specialist</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText">
<p></p>
<div>
<p>We are currently looking for a dynamic individual as an <b>IT Security Specialist </b>for our Global Information Technology organization at our Corporate Headquarters located in The Woodlands, Texas.
</p>
<p>As an IT Security Specialist, you will protect Huntsman computer assets, business systems and data by ensuring solution and design of system implementation meeting security policies, standards and processes. IT Security Specialist will work with the security team to understand how monitoring and scanning our systems to find security vulnerabilities applies, by analyzing the results of these technical evaluations, by providing guidance in remediation efforts, by educating a broader audience in best-practice security procedures, and by continuously improving this process through advanced training and implementation of more effective defensive measures.
</p>
<p></p>
<p><b>What will be expected from you?
</b></p>
<ul>
<li>Conduct technical analysis of Huntsman IT assets and minimize errors and fraudulent use of Huntsman information systems and data through architecture plans and recommendations
</li>
<li>Conduct or participate in IT Security incident investigations. Seek effective ways to prevent reoccurrence
</li>
<li>Works with the Process Control organization to assist them with IT Security issues. (The process control organization manages the computing systems that operate the manufacturing assets.)
</li>
<li>Participate in Huntsman IT Projects
</li>
<li>Assist local staff in remediation efforts of reported vulnerabilities and in global tracking of resolutions
</li>
<li>Establish good working relationships with Local Security Officers, the members of the Directory Services team, Infrastructure supervisors and managers and other technical personnel within infrastructure and system administrators, and the Process Control Organization and vendors of IT Security products
</li>
<li>Develop and maintain IT Security Standards and Procedures for the Huntsman Corporation and guidelines for projects and application development teams to follow.
</li>
<li>Assist the Global and Regional IT Security and Compliance Managers in providing effective communication with the IT organization.
</li>
<li>Evaluate new security technologies for applicability / cost effectiveness in Huntsman and make recommendations to the Global IT Security Manager
</li>
</ul>
<p></p>
<p><b>What are we looking for in the ideal candidate?
</b></p>
<ul>
<li>Demonstrated success in the above key accountabilities and responsibilities
</li>
<li>A minimum of a Bachelor’s Degree in Computer Science or Information Systems with 10+ years specific experience
</li>
<li>Preferred: CISSP, CCNA, Microsoft Certifications
</li>
<li>Excellent understanding of cloud security strategies
</li>
<li>Excellent understanding of networking concepts and operations
</li>
<li>Excellent analytical skills
</li>
<li>Knowledge of Global Data Privacy regulations
</li>
<li>Working knowledge of common operating systems (Microsoft, Unix, Linux and Cisco)
</li>
<li>Working knowledge of Anti-virus (Malware) software operation, updating and cleansing
</li>
<li>Experience in using IT Security tools that survey a network and ethical use of IT Security diagnostic tools
</li>
<li>Possess good interpersonal skills
</li>
<li>Must have a level of proficiency with Internet, Email, and Microsoft programs
</li>
<li>Operational knowledge of Firewalls and Intrusion detection systems (as opposed to conceptual knowledge)
</li>
<li>Good understanding of the IT systems and telecommunications in use by the various businesses
</li>
<li>Experience with scanning and vulnerability scanning tools
</li>
<li>Experience with Forensic investigations and tools
</li>
</ul>
