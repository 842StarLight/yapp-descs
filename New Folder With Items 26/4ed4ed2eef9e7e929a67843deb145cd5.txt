<p>About This Opportunity
</p>
<div><div><div>
<div>The Software Engineer II is responsible for managing all technical aspects of software development for applications including, but not limited to, the development of prototypes, writing code and application program interfaces (API), testing, support and deployment.</div>
</div></div></div>
<p>Responsibilities
</p>
<div><div><div>
<ul><li>Development, testing and support of all the associated applications
</li></ul>
<ul>
<li> Support design and assist in planning team sprints and projects</li>
<li> Support and maintain projects developed by the team and assist in production outages</li>
<li> Develop new features and enhancements on the platform</li>
<li> Test and certify release of code artifacts</li>
<li> 3-5 years of progressive Java development experience including JSE, JEE in enterprise level scalable, distributed and fault tolerant environments, JMS</li>
<li> Experience with Angular 2 development.</li>
<li> Experience with SQL and DBs such as Microsoft SQL Server
</li>
<li> Good understanding of Object Oriented Programming/Design principles</li>
<li> Experience with various platforms like Windows server, Linux/Unix</li>
<li> Experience reading and writing UML (Specifically class and sequence diagrams)</li>
<li> Familiarity with open source application frameworks</li>
<li> Bachelor’s Degree in Computer Science, Engineering or equivalent work experience
</li>
<li> Flexibility to learn on the job and adapt to a changing environment</li>
</ul>
