What You Will Be Doing</p>
<ul>
<li>ChargePoint is looking for an outstanding engineer interested in applying creativity and a broad skill set toward building cutting-edge software for a next generation factory management and test system.</li>
<li>You will work independently on design, implementation, and testing of new systems in a rapidly evolving factory test system.</li>
</ul>
<p>What You Will Bring to ChargePoint</p>
<ul>
<li>Create local and distributed control and management subsystems for power modules, vehicle communications, and user interfaces.</li>
<li>Factory station device test design.</li>
<li>Support of real-time cloud management and remote administration.</li>
<li>Collaborate cross-functionally and globally with hardware and cloud engineering teams to define specifications, features, and APIs.</li>
<li>Participate in design and code reviews, as well as mentor other members of the team.</li>
</ul>
<p>Requirements</p>
<ul>
<li>Experience developing for embedded systems using C/C++, or Python.</li>
<li>Ability to use and program for Linux is a requirement.</li>
<li>Familiarity with BuildRoot, Python3, QT5, git, SQL, and OOP programming.</li>
<li>Exposure in connectivity architecture using Wireless (Cellular, Wi-Fi, Bluetooth and NFC), and wired (Ethernet) interfaces a plus.</li>
<li>Demonstrated ability and willingness to make critical system-level design decisions and drive projects from start to finish with minimal supervision.</li>
</ul>
<p>Preferred</p>
<ul>
<li>Experience with a range of native graphics / display-based products and user interface application services using QT, Python3, and C++.</li>
</ul>
