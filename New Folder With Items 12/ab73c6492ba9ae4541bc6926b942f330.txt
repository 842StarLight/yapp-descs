<h2>Oracle Cloud PaaS Technical Architect</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText">
<div>
<div>The Technical Architect understands the diverse business needs of each client and implements various Oracle Cloud and E-Business Suite applications. Works as a Technical Lead coordinating with onshore and offshore team members to complete project deliverables.</div>
<div></div>
<div><b>Essential Job Functions</b></div>
<div></div>
<div>Focused in PaaS domain - Application development using ADF, JET, SQL, PLSQL, ICS/OIC, Process Cloud service and similar areas in EBS.
</div>
<div></div>
<ul>
<li>Responsible for delivering technical architecture and presenting it to the client, Oracle etc.
</li>
<li>Design and develop Integration solutions involving both Oracle and non-Oracle applications both on-premise and cloud.
</li>
<li>Produce Integration and Technical Architecture Documents to include Conceptual Technical Models, Platform, Performance,
</li>
</ul>
<div>Security, Operational Requirements, Integration and Batch Requirements.
</div>
<ul>
<li>Identify potential risks and problem areas. Develop procedures that provide effective solutions to the problem at hand.
</li>
<li>Be an onsite technical lead and co-ordinate with offshore team to complete project deliverables.
</li>
<li>Serves as advisory-level (technical) pre-sales support and post-sales technical support specialist when needed.
</li>
<li>Train other DAZ resources and guide them as per project needs.
</li>
</ul>
<div></div>
<div><div>
<div><div><div><b>Knowledge, Skills and Experience Requirements</b></div></div></div>
<br>
<div><div><ul>
<li>BS / MS level technical degree or equivalent experience; Computer Science or Engineering background preferred.
</li>
<li>10+ years of strong software development life cycle experience in Oracle domain.
</li>
<li>Focused experience in PaaS domain - application development using ADF, JET, SQL , PLSQL, ICS/OIC, Process Cloud service and progressive experience in similar areas in the EBS.
</li>
<li>Deep expertise and hands-on experience with PaaS- Java Cloud Service, Database Cloud Service, Integration Cloud Service.
</li>
<li>Prior Experience as a technical architect and lead in the EBS and/or Oracle Cloud implementations.
</li>
<li>Familiarity with Cloud Security Technology and best practices (SSO, IAM, Federated)
</li>
<li>Experience in Authentication and Authorization technologies/protocols (OAuth 2.0, OpenID, SAML)
</li>
<li>Proven Software development experience in at least Java, Oracle SQL/PLSQL.
</li>
<li>Familiarity with backup and disaster recovery options and implementation in the cloud.
</li>
<li>Familiarity with industry compliance and security standards that includes PCI DSS, ISO 27001, HIPAA etc.
</li>
<li>Able to demonstrate high level computer skills and knowledge in MS Office, Outlook, Excel
</li>
<li>Excellent verbal and written communication skills
</li>
<li>Excellent analytical capabilities
</li>
<li>Ability to interact effectively within a team and with client(s)</li>
</ul></div></div>
