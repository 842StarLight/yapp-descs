<p><b>It Works! ™ Devops Engineer</b></p>
<br>
<p></p>
<p><b>
Responsibilities:</b></p>
<br>
<p></p>
<p>
The DevOps Engineer will be responsible for implementing and maintaining Continuous Integration and Continuous Deployments for multiple systems that are housed in co-located sites as well as cloud based systems. You are responsible for establishing DevOps practices inside of the company’s Cloud Software teams. This position will also be responsible for setting up and monitoring cloud resources: web services, functions, service bus topics, to API gateways and virtual networks. Logging of events and system usage will be required in order to help support the programming team's development efforts as well as the support of production systems.</p>
