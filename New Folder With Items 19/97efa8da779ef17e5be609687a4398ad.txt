Depending on your preference, you can become a Backend or Frontend Engineer at WorkSpan.
<br><br>
<b>As our Backend Engineer you will:</b><br>
<ul>
<li>Build efficient, scalable, reusable back-end services and RESTful APIs to power our Web and Mobile applications</li>
<li>Develop and maintain code quality standards and best practices.</li>
<li>Enforce standards through regular code reviews.</li>
<li>Write and maintain internal tools to help increase developer efficiency and effectiveness.</li>
<li>Analyse and improve performance by utilising data to oversee product and user engagement (e.g. volumes and activity, key demographics &amp; trends, etc.) and bring to light areas for improvement.</li>
<li>Prepare and declare recommendations, reports, and findings from experimental data all the way up to the executive level.</li>
<li>Deliver Awesomeness : )</li>
</ul>
<b>As our Frontend Engineer you will:</b><br>
<ul>
<li>Deliver clean and performant AngularJS component</li>
<li>Ensure high performance of the application</li>
<li>Write tested, idiomatic, and documented JavaScript</li>
<li>Synchronise with our backend and HTML / CSS team</li>
<li>Refactor existing code</li>
<li>Analyse and improve performance by utilising data to monitor product, user engagement (e.g. volumes and activity, key demographics &amp; trends, etc.), and uncover areas for improvement</li>
<li>Develop and maintain code quality standards and standard methodologies</li>
<li>Deliver Awesomeness : )</li>
</ul>
<b>Skills and Qualifications we are looking for:</b><br>
<ul>
<li>Deep knowledge in web application development (i.e., prior experience in building large scale web applications)</li>
<li>Expertise in web application performance tuning</li>
<li>High level of proficiency in web security (XSS, CSRF, etc)</li>
<li>Experience maintaining large and growing code bases in a fast-paced environment is a plus.</li>
<li>Interest in being up-to-date with the latest and evolving cloud technologies.</li>
<li>For Backend role: Deep understanding of server side programming languages such as Python / Java / C#.</li>
<li>For Frontend role: In-depth knowledge of Javascript and/or TypeScript and AngularJS or Angular.</li>
<li>5+ years of engineering experience</li>
<li>Bachelor or Master's degree in Computer Science</li>
</ul>
