<b>Position Responsibilities</b><ul>
<li>
Support all areas of the enterprise, ensuring all databases run at optimal performance.</li>
<li>
Ensure that industry-wide security standards are properly implemented.</li>
<li>
Develop and optimize complex T-SQL and PL/SQL code.</li>
<li>
Design and develop SQL code, ETL processes and SSRS reports based on provided requirements.</li>
</ul>
<br>
<b>
Minimum Requirements</b><ul>
<li>
10+ years of experience in data integration, data movement and database administration <b>preferred.</b>
</li>
<li>
Proficient in the following RDBMS: SQL Server 2012-2017 (expert), Oracle 11g and 12c RAC (basic), DB2 LUW 11.1 pureScale (basic).</li>
<li>
Experience with SQL Server replication (transactional, snapshot, SQL and Oracle target).</li>
<li>
Proficient in installing, configuring, upgrading and patching all RDBMS listed above, as well as OLTP performance troubleshooting.</li>
<li>
Experience with HA/DR technology for RDBMS’ mentioned above</li>
</ul>
