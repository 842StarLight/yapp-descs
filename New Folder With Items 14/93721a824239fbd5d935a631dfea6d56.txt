<h2>Senior SQL Server Database Engineer</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText">Market America a Global Product Brokerage and Internet Marketing company that specializes in One-to-One Marketing, is seeking a strongly experienced Senior SQL Server Database Engineer for our IT environment.

Position Summary
<br><br>
The Sr. SQL Server Database Engineer will work on development, implementation, delivery and maintenance of Microsoft SQL Server and other related SQL based technology systems for Market America and SHOP.COM Worldwide. This includes responsibility for SQL database development and administration for extensive custom applications supporting business operations, and for operational reporting and data warehouse analytics. The Engineer will ensure the smooth and effective operation and 24/7 availability of the systems.
