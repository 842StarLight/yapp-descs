<h2>Developer Tools Engineer</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText">Yesware helps our users spread the adoption of new ideas, person to person. We build innovative software that follows salespeople through their day, enhancing their productivity, helping them be more authentic, and delivering data-driven recommendations that help them win business. We are building out a Developer Tools function to enhance our talented development org.
<br><br>
As a Developer Tools Engineer at Yesware, you will be accelerating product development by making our engineers more productive. You will utilize a data-driven approach to measure and improve our developer throughput. You will create a development environment that engineers love to use because it is easy to work with and gives developers confidence. You will reduce test cycle time, make it easy to add new tests, improve our sandbox and staging environments, and help engineers design for testability. This role provides ample opportunity to apply previous experience, develop new skills, and be a part of a growing company.
<br><br>
Yesware has a feature-rich backend architecture and data platform, along with apps for Gmail, Outlook, and the web, which collectively serve over 100K users worldwide. We run a Javascript and Ruby on Rails stack hosted on the Heroku PaaS. Our systems are disconnected macro- and microservices communicating via REST APIs and asynchronous message queues. We use PostgreSQL, MongoDB, Redis, RabbitMQ, and Snowflake to store and communicate customer data at scale.
<br><br>
<b>Responsibilities:</b><br>
<ul>
<li>Challenge us to find more efficient ways to develop, test, and deploy</li>
<li>Anticipate and proactively address blockers before they become widespread</li>
<li>Improve engineering productivity using a metric-driven approach -- we’re looking to you to establish metrics, monitor, and iterate</li>
<li>Extend or replace existing development and deployment tools</li>
<li>Develop, enhance, and review test automation tools, frameworks, and infrastructure</li>
<li>Automate as much as possible</li>
</ul>
<b>Requirements:</b><br>
<ul>
<li>Experience implementing development tools such as version control systems, build or test systems, deployment utilities, etc.</li>
<li>Experience in Ruby, Python, Java, or similar environments</li>
<li>Experience with software development in a Unix-based environment and writing/maintaining shell scripts, preferably in Bash</li>
<li>Excellent software development habits (object-oriented design, unit testing, integration testing, data structures, etc.)</li>
<li>Excellent oral and written communication skills</li>
<li>Emotional maturity and adaptability</li>
</ul>
<b>Nice-to-Haves:</b><br>
<ul>
<li>Experience working with a large, high-scale web application (well-designed APIs, high volume data pipelines, efficient algorithms, etc.)</li>
<li>Experience with developer technologies such as TeamCity, GitHub, Docker, Homebrew, Ansible</li>
<li>Experience working with SQL and/or NoSQL databases at scale</li>
<li>Experience supporting cloud-hosted SaaS applications in a DevOps capacity</li>
<li>Experience developing SaaS offerings</li>
</ul>
