<h2>Software Engineer</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText">
<p><b>Typical Project Responsibilities: </b></p>
<ul><li>Collaboratively prioritizing requirements with Customers, Sales, Marketing, Service and Manufacturing</li></ul>
<ul>
<li>Determining the architecture and detailed design approach</li>
<li>Documenting project requirements</li>
<li>Executing and reviewing designs</li>
<li>Developing test plans and procedures for the project, ensuring they are executed correctly and any deviations are addressed</li>
<li>When applicable, understand present code base and product use model</li>
<li>Other duties as assigned</li>
</ul>
<p><b>Required Skills: </b></p>
<ul>
<li>Solid understanding of software algorithms and data structures</li>
<li>Object oriented design skills including object design and responsibility assignment, low coupling and high cohesion, and polymorphism</li>
<li>Demonstrated skills in C/C++ and Java or C#</li>
</ul>
<p><b>Highly Desired Skills: </b></p>
<ul>
<li>Experience in mobile development using Android Studio</li>
<li>Mobile UI development skills</li>
<li>Experience with both Java and C threading</li>
</ul>
<p><b>Desired Skills: </b></p>
<ul>
<li>Experience using a Realm object database in a mobile environment</li>
<li>Experience using SciChart for creating charts and graphs in a mobile environment</li>
<li>Experience developing in C in a Linux environment</li>
<li>Experience developing for an embedded system</li>
<li>Experience using JNI to interface between native C libraries and managed Java to provide low level support to Java applications</li>
</ul>
<p><b>Experience and Qualifications: </b></p>
<ul>
<li>Bachelors or Masters degree in Computer Science, or equivalent experience</li>
<li>2-5 years’ experience in a software engineering role</li>
<li>Experience in engineering of products consisting or electronic, software, and mechanical content, with history of meeting schedule, scope and budgets.</li>
</ul>
<p><b>Additional Knowledge/Skills/Abilities: </b></p>
<ul>
<li>Technical expertise and the knowledge to apply technology effectively</li>
<li>Is a self-starter – takes initiative to make things happen</li>
<li>Continually improves self, product, process and team</li>
<li>Innovates in the areas of product concepts, project content and technology application</li>
<li>Satisfies customers, both internal and external</li>
<li>Demonstrates high level of professional integrity</li>
</ul>
