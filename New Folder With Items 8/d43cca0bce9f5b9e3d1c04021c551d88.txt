<b>Purpose:</b><br>
To provide senior level software design and development for a family of multi-tier 24x7 Azure Cloud hosted web applications that scale to a large number of web users and large backend databases. The applications focus on the consumer engagement through the analytics of utility usages such as billing data for electric, gas, and water consumption. Develop applications in Microsoft .NET technologies, SQL Server database, Cassandra NoSQL database, Azure Cloud Platform services, Azure Data storage, Web APIs, CSS, and JavaScript frameworks, such as Bootstrap and AngularJS. As a senior full stack developer, you will work in a scrum team. You will be responsible for design, development, testing, maintenance, and enhancements of the ACE applications written in high level languages (C#, AngularJS, T-SQL, CQL). In addition, this role includes designing, coding, and testing for web application security, performance, and scalability.
<br><br>
<b>Essential Functions:</b><br>
<ul>
<li>Work closely with SMEs and Product Management to validate functional and non-functional requirements to satisfy the product use cases.</li>
<li>Troubleshoot and resolve production issues. This requires you to work with cross-functional teams such as Professional Services, Support, and Sales.</li>
<li>Author and review artifacts of all phases of a project, such as: use cases, estimates, high- and low-level designs, code, and test cases to make sure they meet requirements. This requires working with the architect to ensure that product architecture is cohesive and adhering to the required functional and non-functional requirements like performance, scalability, and security.</li>
<li>Review new product requirements, gap analysis provided by professional services, then work with the Pod leader and the Product Owner to create user stories to address these requirements and use cases.</li>
<li>Ensure software quality responsibility through demos, creation of solid testing plans, execution and reporting on the test results.</li>
<li>Serve as technical conduit responsible for ensuring clear communication, solid commitments, and clear accountability by all groups involved: Project Management, Product Management, Software Engineering, and the Professional Services team.</li>
<li>Mentor junior engineers and provide them with technical oversight.</li>
</ul>
