<h2>Senior RF Hardware Test Engineer</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText"><div>
<p>Senior RF Hardware Test Engineer</p>
<p><b>
Location: </b>San Jose, CA</p>
<p></p>
<br>
<p><b>
Position Summary:</b></p>
<p>
Samsung Semiconductor, Inc. is a world leader in Memory, Display and recently formed Mobile and Connectivity technology. We are looking for a Senior RF HW Test Engineer to design and conduct automated testing for the Location team. The ideal candidate will have at least 3 years’ experience in the design of RF HW and automated test systems.</p>
<p></p>
<br>
<p><b>
Common Essential Duties &amp; Responsibilities:</b></p>
<ul>
<li>
Assist the SW Test Engineer with the existing software code and recommend changes to simplify the test method in order to improve testing efficiency.</li>
<li>
Work with HW and SW engineers to resolve any issues that may occur during HW/SW integration.</li>
<li>
Help engineers with test setup, conducting tests to collect data and generation of test reports.</li>
<li>
Document the auto test methodology.</li>
<li>
Verify chip performance through lab and field tests</li>
<li>
Make measurements using instruments such as Oscilloscope, Signal generators, Spectrum Analyzer, VNA etc.</li>
<li>
RF board level troubleshooting</li>
<li>
Train test technicians.</li>
</ul>
<p></p>
<br>
<p><b>
Background/Experience:</b></p>
<ul>
<li>
Bachelor’s in Electrical Engineering, Masters preferred.</li>
<li>
RF background with minimum 3 year experience.</li>
<li>
Experience with software such as Python.</li>
<li>
Must understand basic RF measurements and lab test equipments.</li>
<li>
</li>
</ul>
<p><b>
Necessary Skills/Attributes:</b></p>
<ul>
<li>
Basic understanding of GNSS test equipment.</li>
<li>
Basic understanding of RF concepts and measurements such as Noise, Noise Figure, Phase Noise, Path Loss, and receiver system design aspects.</li>
<li>
Experience with HW validation testing is required.</li>
<li>
Excellent analytic problem solving and debugging skills.</li>
<li>
Solid troubleshooting skills at the component and system level.</li>
<li>
Ability to multi-task and easily adapt within a fast-paced environment.</li>
<li>
Must be open for occasional international travel (up to ~3 trips a year)</li>
<li>
</li>
</ul>
