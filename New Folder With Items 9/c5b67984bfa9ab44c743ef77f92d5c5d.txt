<h2 class="jobSectionHeader"><b>Responsibilities &amp; Qualifications
</b></h2>
<p></p>
<p>The Back-end Lead Application Developer performs complex systems analysis and programming tasks requiring advanced techniques. Evaluates technical feasibility of proposed system solution. Leads projects and directs the work of other developers as well as provides training for less experienced staff. Developer stays current on new and emerging technologies.
</p>
<p></p>
<p><b>RESPONSIBILITIES
</b></p>
<ul>
<li>Define service specification and contracts for a functional domain by working with SMEs
</li>
<li>Strong technical, analytical, and communication skills
</li>
<li>Collaborate with Architects and contribute to implement standards and changes to systems and processes
</li>
<li>Ensuring design and development follows enterprise standards, processes, and policies
</li>
<li>Support application deployment and release management activities
</li>
<li>Collaboratively work with other groups by offering technical expertise where required
</li>
<li>Mentor SOA developers, other team members
</li>
<li>Leading maturity and transformation of platforms, developing new capabilities
</li>
<li>Supporting multiple concurrent efforts
</li>
<li>Participate in business requirement discussions, identify system impacts and provide business requirements, system requirement specifications.
</li>
<li>Be a team player that motivates and educates other team members
</li>
</ul>
<p></p>
<p><b>ESSENTIAL SKILLS &amp; EXPERIENCE
</b></p>
<ul>
<li>College degree in Information Technology or Computer Science
</li>
<li>Minimum of 8+ years’ work experience in Information Technology.
</li>
<li>8+ years object-oriented software development/architecture experience in JEE
</li>
<li>Solid Programming skills using JAVA, J2EE, JMS, SOAP / REST Web Services, XSLT, XML, JSON.
</li>
<li>Experience with Spring MVC, Spring boot
</li>
<li>Good experience in integration Architecture concepts and patterns, including Microservices, Service Oriented Architecture, Batch Integration, RESTful JSON services, Pub / Sub etc. including Event Driven Architecture.
</li>
<li>Experience of database access protocols such as JDBC and SQL
</li>
<li>Experience integrating with Cloud/SaaS applications, APIs, SDK of packaged applications and legacy applications
</li>
<li>Solid understanding of integration design patterns, SOA/EAI best practices.
</li>
<li>Knowledge of VSTS / CICD Process.
</li>
<li>Experience with Test Driven Development and JUnit.
</li>
<li>Experience with SCRUM or similar agile SDLC methodologies
</li>
<li>Proven experience in meeting deadlines and managing multiple priorities.
 </li>
</ul>
