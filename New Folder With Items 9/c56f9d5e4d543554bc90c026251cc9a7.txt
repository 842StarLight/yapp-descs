<h2>DevOps Engineer III</h2> 
 <div id="jobDescriptionText" class="jobsearch-jobDescriptionText"><div>
<div>
<p><b>Position/Title: </b>DevOps Engineer III (Azure/TFS/VSTS) - 100% Remote</p>
<br>
<p></p>
<p><b>Location: </b>100% Remote</p>
<br>
<p></p>
<p><b>Primary Purpose:</b></p>
<p>Responsible for the design and development of builds, scripts, installation procedures, and systems including source code control and issue tracking. Provide communication and collaboration between software development and infrastructure team.</p>
<p><b>Essential Duties and Responsibilities:</b></p>
<ul>
<li>Develop and implement .NET application software release practices that can be used enterprise wide to create a consistent and commonly understood process by all concerned parties</li>
<li>Collaborate with developers and establish Visual Studio solution organization</li>
<li>Setup Azure DevOps Services builds, transformations and deployment configurations for each environment</li>
<li>Setup continuous integration capabilities</li>
<li>Delegate access to application developers to trigger non-production builds/deployments</li>
<li>Perform production build/deploy activities for application releases</li>
<li>Support production deployments, assist with troubleshooting deployment issues</li>
<li>Assist with branching/merging in support of parallel development requirements</li>
<li>Collaborate with testing teams and assist with performance testing setups/executions</li>
<li>Facilitate load/performance testing in support of business sales cycles</li>
<li>Liaison between application development and infrastructure teams in the establishment of needed infrastructure in the development and subsequent support of application solutions</li>
<li>Develop an understanding of infrastructure footprint of existing .NET solutions and utilize as a pattern for new solutions</li>
<li>Serve as the SME for software development life-cycle tools – tools support testing, monitoring, integration/release and scheduling areas</li>
<li>Work closely with internal teams such as Development, Infrastructure, DBAs, and Storage</li>
<li>Install, configure, troubleshoot, patch, and upgrade software application products</li>
<li>Monitors usage and changes configurations to improve capacity or performance of servers</li>
<li>Participate in system performance tuning and technical troubleshooting sessions</li>
<br>
</ul>
<p></p>
<p><b>Minimum Skills and Competencies:</b></p>
<ul>
<li>Bachelor degree in Computer Science or Computer Information Systems</li>
<li>Strong hands on experience in CI/CD implementation within Azure DevOps Services (formally Visual Studio Team Services)</li>
<li>4 - 6 years of distributed build/deploy experience</li>
<li>Heavy hands on experience in Azure DevOps Services (formally Visual Studio Team Services) GIT experience, including build and release pipeline automation, branching and merge strategies.</li>
<li>Strong knowledge of iterative Software Development methodology including design, coding, testing and implementation.</li>
<li>Proficient in Microsoft Office (Word, Excel, Outlook, PowerPoint)</li>
<li>Performs work under minimal supervision</li>
<li>Handles complex issues and problems, and refers only the most complex issues to higher-level staff</li>
<li>Ability to lead and work with other team members, including mentoring less experienced team members</li>
<li>Ability to work with and understand business client requirements</li>
<li>Excellent interpersonal skills with the ability to establish working relationships with individuals at varying levels within the organization.</li>
<li>Effective organizational and time management skills with the ability to work under pressure, adhere to project deadlines and procedures, multi-task and be detail-oriented</li>
<br>
</ul>
<p></p>
<p><b>Desired Skills:</b></p>
<ul>
<li>Strong knowledge of scripting languages – Powershell preferred.</li>
<li>Strong knowledge of server technologies (ex. IIS configurations, monitoring agents) and network configurations (ex. DNS, proxy, firewall rule exceptions)</li>
<br>
</ul>
