<p><b>
The Opportunity:</b></p>
<p></p>
<p>
Ideal candidate is a strong self-driven individual with high level of competence in a variety of technical backgrounds to include networking, systems, security and automation. This person should be highly technical with ability to operate autonomously and employ cross-team collaborate to solve complex and challenging problems.</p>
<ul>
<li><p>
Responsible for the reliability, availability and performance of production data-center networks.</p></li>
<li><p>
Respond to network trouble ticket escalations from GSNOC, analyzes and solves reported network problems. Interacts with Infrastructure and Ops teams to provide issue resolution.</p></li>
<li><p>
Principal resource for design and implementation of L7 application delivery, routing and load balancing solutions for production apps and services.</p></li>
<li><p>
Collaborate with network and security architects to review, and approve, the alignment of new project initiatives to company security standards. Actively participates in the development of security standards.</p></li>
<li><p>
Plan and execute zero downtime hardware and software upgrades of in-service production network equipment in live traffic scenarios.</p></li>
<li><p>
Implement network changes in support of project initiatives.</p></li>
<li><p>
Automate management of network devices with Python.</p></li>
</ul>
<p></p>
<p><b>
Required Skills</b></p>
<ul>
<li><p>
Operational experience in large network support with a data center focus</p></li>
<li><p>
In-depth knowledge of routing, switching and load balancing protocols and technologies</p></li>
<li><p>
Good understanding of how DNS works</p></li>
<li><p>
Experience managing CDNs (Akamai, Cloudflare, etc).</p></li>
<li><p>
Experience designing secure networks, systems and application architectures</p></li>
<li><p>
Intimate knowledge of the HTTP protocol</p></li>
<li><p>
Proficient in at least one scripting language (Python, Bash, etc)</p></li>
<li><p>
Basic understanding of Linux</p></li>
</ul>
<p></p>
<p><b>
Desired Skills</b>:</p>
<ul>
<li><p>
F5 BigIP (LTM, GTM, APM, ASM)</p></li>
<li><p>
Cloud networking and security knowledge, focus on GCP and Kubernetes</p></li>
<li><p>
Familiarity with at least one monitoring/logging tool (e.g. Splunk, OP5, Nagios or DynaTrace)</p></li>
<li><p>
Akamai, Cloudflare or similar CDN</p></li>
</ul>
<p></p>
<p><b>
Relevant skill areas</b>: CCNA/CCNP or JNCIS/JNCIP, CCNP Security, CEH, GIAC, JUNOS, F5 BigIP, OSPF, IS-IS, BGP, DNS, TCP/IP, HTTP</p>
<p></p>
<p><b>
Roles and Responsibilities</b></p>
<ul>
<li><p>
Troubleshoot network issues in production data-centers</p></li>
<li><p>
In-place hardware upgrades for vital networking equipment with live traffic</p></li>
<li><p>
Manage ACLs for firewalls and network proxies</p></li>
<li><p>
Write F5 iRules to intelligently route traffic at layer 7 in various scenarios.</p></li>
<li><p>
Manage PKI for SSL certificates</p></li>
<li><p>
Conduct network/security vulnerability analyses and risk assessments</p></li>
<li><p>
Automate configuration changes to network devices</p></li>
<li><p>
Manage DNS for production sites</p></li>
<li><p>
On Call rotation support</p></li>
</ul>
