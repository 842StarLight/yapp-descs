<p><b>Role Summary / Purpose</b></p>
<p>Alstom has an opening for a Real-Time Software Engineer, in its Melbourne, FL location.</p>
<br>
<p></p>
<p>This position is for a highly motivated, energetic Software Engineer who will develop real-time embedded software for our CORE wayside systems.</p>
<br>
<p></p>
<p>This position is for a self-motivated, confident, open source savvy individual. The successful candidate will demonstrate the ability to execute as an individual developer and an individual contributor in a team. The Software Engineer will be experienced in real-time and embedded programing techniques in C/C++ on a standalone bare metal or RTOS target using an Agile process.</p>
<br>
<p></p>
<p><b>The successful candidate will be responsible for:</b></p>
<ul>
<li>Working closely with Systems Engineers to understand operations and translate them into clear software requirements.</li>
<li>Designing, documenting, implementing and unit testing C/C++ embedded real-time software.</li>
<li>Providing input to software deliverables such as software requirements specifications, software design documents, and professional high quality software executables.</li>
<li>Developing safety critical software using vital software development standards.</li>
<li>Communicating schedule updates, risks, and plans to software technical leaders and other stakeholders</li>
<li>Working with field support teams to diagnose customer issues and quickly resolve issues</li>
<li>Executing Quality Management Standards and necessary vital software design standards</li>
<br>
</ul>
<p></p>
<p><b>Qualifications:</b></p>
<ul>
<li>Bachelor's Degree in Electrical Engineering, Software Engineering, Computer Science, Computer Engineering or other related engineering degree</li>
<li>Real-Time Software Development Engineering experience in C / C++</li>
<li>Experience developing software in a cross platform development environment (Eclipse, GreenHills, etc.)</li>
<li>Experience in Agile Software Development Process using software development life cycle tools (git, JIRA, and Confluence).</li>
<br>
</ul>
<p></p>
<p><b>Desired Qualifications:</b></p>
<ul>
<li>Network programming with exposure to IP protocol stacks (TCP / UDP / FTP)</li>
<li>Safety Critical Systems and Software Design</li>
<li>Rail Domain Signaling Systems: ITCS, Cab-Signaling, CBTC, IETMS, PTC</li>
<li>Experience in requirements management tool of DOORS</li>
<br>
</ul>
